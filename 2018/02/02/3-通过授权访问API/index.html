通过授权访问API

在上一节中，我们已经成功将后台API保护了起来，但我们应该如何访问该API呢？这节我们将通过一个简单的Console例子，来实现认证并获取资源的流程。

构建Client

首先还是一样，我们在VS中新建一个.net core Console项目，并为其添加IdentityModel的引用。

虽然微软已经在最新版本的C#中，添加了Main函数可以异步使用的特性。但考虑到大部分人使用的版本并不是最新的，这里还是先把Main函数改造一下，变成如下形式。

首先利用DiscoverClient 从metadata中获取 EndPoint。

然后，根据第一节中Client的注册信息来创建Client，这里我们使用下面两种Client。

  1. ClientCredentials
    • ClientId: client
    • ClientSecret: secret
  2. ResourceOwnerPassword
    • ClientId: ro.client
    • ClientSecret: secret

ResourceOwnerPassword的用法和上述相似，只是需要在请求token时提供user和password。
